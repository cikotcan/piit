# mtcars Analysis Project

## Objective
To analyze the mtcars dataset using Python and Pandas.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Matplotlib
- GitHub

## Tasks Performed
- Data loading and inspection
- Groupby analysis
- Pivot table creation
- Data mapping
- Visualization
- Insights and conclusion

## Key Findings
- Cars with fewer cylinders have better mileage
- Lighter cars have higher MPG
- Manual cars often perform better
